Issue legal receipts automatically directly from your wordpress using MyData provider Services.

== Description ==

Convert your e-shop orders to legal receipts and invoices directly from your wordpress website! No external ERP, fiscal machine or cash register is required!
Only for Greek e-shops
Works with woocommerse
Compatibility with other e-shops coming soon!

**Needs an active MyData subscription service to work. [Get MyData subscription here](https://primer.gr/plugin/)**

* Easy conversion of woocommerse orders to receipts/invoices
* No MyData setup is needed - Works out of the box!
* Free up time and cost from having to issue receipts/invoices manually
* No external ERP/fiscal machine/Cash Register required!
* No complicated connections with external software
* Plugin automatically understands and issues the correct document (receipt/invoice/invoice within E.U./invoice outside E.U.

= Features =

* Plugin is free (Needs MyData service to work)
* Orders management
* Receipts Management
* Issue Receipts/Invoices/Invoices within E.U./Invoices outside E.U.
* Manual conversion of orders to receipts/invoices
* Automatic conversion of orders to receipts/invoices based on rules (when order status is "x", issue receipt)
* Connection to Primer MyData services
* Automatically send receipt/invoice to email to the client using Primer Email servers
* Checkout options (Receipt/Invoice)
* Smtp options and Email builder - send the receipt your way!
* Automatically export receipts/invoices to email/server
* Connection with external software using export can be automated
* Automatic MyData characterizations
* Invoice/Receipt templates
* Test environment to emulate the sending to AADE before sending to production environment
* Easy to use interface

== Frequently Asked Questions ==

= Can I issue legal receipts with this plugin? =
Yes, you can issue legal receipts with this plugin, because it's using the Primer MyData services provider, whom is a licenced MyData provider from AADE. You can check the licenced providers at https://www.aade.gr/adeiodotimena-logismika-parohon-ilektronikis-timologisis

= Where can I find documentation for Primer MyData plugin? =
You can find the documentation at https://primer.gr/plugin/
